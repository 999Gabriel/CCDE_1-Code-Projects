# Millionaire Game Clients

This directory contains client applications for playing the Millionaire Game.

## Prerequisites

Ensure the game server is running:

```bash
# In the project root
python app.py
```

## Clients

### 1. Node.js TUI Client (`client_node.js`)
A Text User Interface (CLI) client featuring ANSI colors and full game loop.

**Usage:**
```bash
node client_node.js
```
*Requires Node.js v18+*

### 2. Python GUI Client (`client_gui.py`)
A graphical client using Tkinter (if available).

**Usage:**
```bash
python client_gui.py
```
